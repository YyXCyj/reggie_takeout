# reggie_takeout

>>本项目是专们为餐饮企业定制的一款软件产品，包括系统管理平台和移动端应用两部分。
>>
>>其中系统管理后台主要提供给餐饮企业内部员工使用，可以对餐厅的餐厅的菜品、套餐、订单等进行管理维护。
>>
>>移动端应用主要提供给消费者使用，可以在线浏览菜品、添加购物车、下单等。
